Lesson 1: Basics of the the World Wide Web

The Web

The Web is a collection of Documents written in HTML. HTML stands for HyperText Markup Language, and is the basis for each document on the web. The web works by your browser requesting a document on a server over HTTP. This is done either via the internet or a local network.

HTML

HTML is made up of text content, markup(how its arrianged), audio/video and links. Markup is made with the follow syntax: <name of tag>Contents</name of tag> In some cases tags tags may be void as they have no content. An example of this would be the code used to insert a picture into an html document

Attibutes

Attributes are items that can be attached to tags. The syntax is as follows: <name of tag attribute ="value">Contents</name of tag>

Inline vs Block Elements

Inline and Block is a way to format text on a document. A block element takes up the width and height available. Block elements provides line breaks before and after the content. This can be further customized by specifing height and width. Inline is simply a line break.

Span vs DIV

Span is a inline element. Div is a block element. Both have a similar syntax: A Span is expressed as: <span class="foo">Content</span>. While a DIV is expressed as: <div class="bar">Content</div>

Webpages, Documents and Structure

HTML documents have a structure that can be described like a tree. Within the HTML tag (Trunk), you can create subtags (Branches) to define additonal form of your document. This structure is governed by the Document Object Model, or DOM. The DOM provides a API for creating a structured, tree like model used in HTML.

HTML-CSS-DOM

  1. HTML - HTML is the language of the Web. HTML has a syntax and rules on how it is displayed.
  2. CSS - CSS is the syntax and rule of how content is displayed on a document.
  3. DOM - Document Object Model

Lesson 3: Adding CSS Style & HTML Structure

Cascading Style Sheets

Cascading Style Sheets or CSS is the syntax and rule of how content is displayed on a document. They work at a broad page level and get more specific to each element. CSS can be applied inline to the element, linked to from the HTML, or builtin to the document head.

Inheritance

CSS has a Parent Child relationship with child objects. Child objects inherit text related properties, but not box related properties. Items that can be inheritied in the CSS chain include:

  1. Color
  2. Font
  3. Space of Characters
Selector

A Selector is a element on the page in which style will apply to. The Syntax of CSS is: Element of Class{style} When applying style the syntax is: nameOfAttribute: Value;. This makes up the decleration.

Box Revisited

Boxes include the following:

  1. Margin - Clears area around the border
  2. Padding - Clears the area around the content. This is the space inside of the box that protect the content.
  3. Border - Applied around the Padding and Content
  4. Content - Image or text

Sizing can be applied to boxes. A best practice is to use Precentage rather than Pixels.
